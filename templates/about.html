Система учета личных финансов

Данный проект предоставляет следующие возможности:

  • Регистрация нового пользователя
  • Вход в аккаунт
  • Управление бюджетом пользователя
  • Добавление транзакций (расходов)
  • Просмотр списка транзакций
  • Получение информации о текущем бюджете пользователя
  • Удаление аккаунта с удалением всех связанных транзакций

Для использования API и тестирования возможностей проекта, вы можете использовать программы для работы с HTTP-запросами, например, Postman.

Для начала работы с проектом, перейдите на главную страницу.
